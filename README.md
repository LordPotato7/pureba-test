## Aprendizaje

Aprendi que para realizar un buen test hay que:

Preparar (Arrange): Esto significa configurar el estado incial.

Actuar (Act): Se refiere a ejecutar la operacion a probar.

Afirmar (Assert): Verifica el resultado
